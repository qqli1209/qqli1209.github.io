用 CSS 画三角形和平行四边形

发表于 2017-03-17 浏览 - 次 分类于 前端学习 | CSS

用 CSS 做了三角形和平行四边形，三角形还可以拖放。

三角形

从下面几个图形你应该可以大概看出来三角形是怎么一步步画出来的。第一个图形 width,height和border分别设置为 50px, 50px 和 25px。第二个图形把width和height设置为 0。第三个图形把右、下、左border的颜色设置为 transparent, 这样看上去就只剩一个三角形啦。

可拖放的元素

拖放是一种非常流行的用户界面模式。它的概念很简单：点击某个元素，按住鼠标不放，将鼠标移到另一个区域，然后释放鼠标，元素就停留在了另一个区域。这需要设置position: absolute并为 mousedown, mousemove, mouseup 绑定相应的事件。

这里我使用自定义事件来提示这几个事件的发生，让应用的其他部分与应用功能进行交互。因为自定义事件有助将不同部分的让代码之间相互解耦，使得维护更加容易。

平行四边形

平行四边形实际就是利用transform: skew(...)把矩形倾斜一下。然后再为里面的文字设置相反的倾斜角就 ok 了。
